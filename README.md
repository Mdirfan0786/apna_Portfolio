## ⚙️ Installation & Setup

Clone the repository:

```bash
git clone https://github.com/Mdirfan0786/apna_Portfolio.git
```

Navigate to project directory:

```bash
cd apna_Portfolio
```

Install dependencies:

```bash
npm install
```

Run development server:

```bash
npm run dev
```

Build for production:

```bash
npm run build
```
